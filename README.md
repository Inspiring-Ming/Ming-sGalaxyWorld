## Customization

### Update Content

- **Photo**: Replace the photo placeholder in the About section with an actual image
- **Publications**: Add more publications from your Google Scholar profile
- **Projects**: Add more project cards with your work
- **Experience**: Fill in earlier career and education timeline items
- **Blog**: Add actual blog posts when ready (currently showing placeholder "Coming Soon" cards)

### Edit Styles

- Colors and theme: Modify CSS variables in `:root` at the top of `style.css`
- Layout: Adjust grid layouts in respective sections
- Fonts: Currently using Inter (body) and JetBrains Mono (code) from Google Fonts
