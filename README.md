# Personal Website - Mingqin (Jasmin) Yu

A modern, interactive personal portfolio website showcasing research publications, projects, and professional experience.

## Features

- Single-page application with smooth scrolling navigation
- Responsive design (mobile, tablet, desktop)
- Modern dark theme with gradient accents
- Sections: About, Publications, Projects, Experience, Skills, Blog, Contact
- Animated scroll effects and interactive elements
- GitHub Pages ready

## Live Site

Will be hosted at: **https://Inspiring-Ming.github.io** (after deployment)

## Project Structure

```
personal-website/
├── index.html      # Main HTML structure
├── style.css       # Styling and responsive layout
├── script.js       # Interactive features and animations
└── README.md       # This file
```

## Local Development

1. Open `index.html` in a browser
2. No build process required - pure HTML/CSS/JS

## Deployment to GitHub Pages

1. Create a new GitHub repository named `Inspiring-Ming.github.io`
2. Push this code to the repository:
   ```bash
   git remote add origin https://github.com/Inspiring-Ming/Inspiring-Ming.github.io.git
   git branch -M main
   git push -u origin main
   ```
3. GitHub Pages will automatically deploy from the `main` branch
4. Site will be live at `https://Inspiring-Ming.github.io`

## Customization

### Update Content

- **Photo**: Replace the photo placeholder in the About section with an actual image
- **Publications**: Add more publications from your Google Scholar profile
- **Projects**: Add more project cards with your work
- **Experience**: Fill in earlier career and education timeline items
- **Blog**: Add actual blog posts when ready (currently showing placeholder "Coming Soon" cards)

### Edit Styles

- Colors and theme: Modify CSS variables in `:root` at the top of `style.css`
- Layout: Adjust grid layouts in respective sections
- Fonts: Currently using Inter (body) and JetBrains Mono (code) from Google Fonts
