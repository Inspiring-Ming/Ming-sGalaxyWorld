ESG Knowledge Graph System

An ontology-driven ESG metric system built on SASB standards. Uses an RDF knowledge graph (525 triples) to map reporting frameworks, categories, metrics, and calculation models, then computes ESG metrics from real company data (422K+ records across 40+ companies).

  • 4-service architecture: Knowledge Graph, Data Retrieval, Calculation, Report Generation
  • 7 competency questions (CQ1–CQ7) answered via SPARQL
  • PDF and Word report generation with data lineage
  • Covers Semiconductors (TC-SC) and Commercial Banking (FN-CB)

OntoMetric Framework

An ontology-driven LLM-assisted framework for automated ESG metric knowledge graph generation. Three-stage pipeline (Segmentation → Extraction → Validation) achieving 65–90% semantic accuracy across SASB, TCFD, IFRS S2, and AASB S2 frameworks with 48× cost reduction vs baseline.

  • Ontology-guided extraction: 5 entity types, 5 relationship types
  • Two-phase validation: semantic type verification + rule-based schema checking
  • Evaluated on 5 regulatory documents: 364 entities → 295 validated (81% retention)
  • 80%+ schema compliance, 63–90% relationship retention, full provenance tracking
